¿Qué layout prefieres para el Login / Register?

Las 3 opciones usan el mismo sistema de colores del proyecto (slate-900 + brand-700 + accent naranja)

A — Card centrada

Fondo oscuro full-screen, card flotante al centro. Clásico y limpio.

B — Split pantalla

Mitad izquierda con branding/imagen, mitad derecha con el formulario.

C — Página clara con marca

Fondo claro (#f8fafc), logo arriba, card blanca. Más corporativo/neutro.
